# Week 3: Multi-Source Data Pipeline

This project demonstrates a multi-source data integration pipeline built using Python. It combines a cleaned internal operational dataset, live weather data from the OpenWeatherMap API, and maintenance records stored in a SQLite database to create a unified dataset for operational analysis.

The project explores how external factors, such as weather conditions, can provide additional context for understanding changes in operational throughput and support better business decision-making.

## Technologies Used

* Python
* Pandas
* Requests
* SQLAlchemy
* SQLite
* Matplotlib
* Jupyter Notebook

## Key Features

* Multi-source data integration
* REST API data retrieval
* SQL database querying
* Data merging and cleaning
* Exploratory data analysis
* Operational insights and recommendations
